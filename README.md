# Farooq Khan Ayub Khan — Transport & General Contracting

Official website for **Farooq Khan Ayub Khan Transport & General Contracting – L.L.C – S.P.C**, Abu Dhabi, UAE.

A responsive, animated single-page site (black & gold luxury theme) covering the company's
luxury chauffeur fleet, passenger transport, heavy equipment, cranes and general contracting services.

## Structure
- `index.html` — the website
- `css/styles.css` — styles
- `js/main.js` — interactions (scroll reveal, slideshow, nav, counters)
- `assets/` — images
- `Farooq-Khan-Profile.html` — printable company profile (PDF-ready)
